new Model()

The Model class is inspired on the Backbone.Model class, with setters, getters, pubsub and a very basic AJAX layer (JSONP).

Example
var model = new Model({
  recorderId: 'abc',
  duration: 30,
  H264Recording: true
});

model.set('net_connection', 'success');
model.get('duration') // 30
